# Advanced Research Computing Services

Advanced Research Computing Services (ARCS) is a division within the Information Services & Technology. 

*To  empower the research and teaching community at NJIT by providing advanced computing resources, fostering technological expertise, and promoting innovation through reliable infrastructure and collaborative support*. 

## High Performance Research Computing

Provides access to High Peformance Computing Cluster for computational and data enabled research. More details available at https://hpc.njit.edu.

##  Research Technology

Support Linux based (not HPC) servers and workstations for research. We work closely with the Core Systems and Cloud Computing group facilitating access to technologies for research and academic use such as 

 - data storage
     - Highlander AFS
     - NetApp 
 - cloud computing resource,
     - database servers 
     - web servers
 - github enterprise cloud
